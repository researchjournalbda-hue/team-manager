The schedule comes from the Run Sheet tab: the page reads each activity's timing, room, description and responsibilities, and writes the people you assign into Team Member and the completion state into Status when you Save. It pulls the latest shared state on open; Refresh from Sheet re-pulls it. Save never emails — Push is the only thing that sends calendar invites, and re-pushing cancels any invite that no longer applies so nothing duplicates.
